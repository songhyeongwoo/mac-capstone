## 프로젝트 소개

MAC Capstone은 데이터 시각화에 익숙하지 않은 사용자가 CSV 또는 Excel 파일을 업로드한 뒤, 데이터 확인과 차트 선택, 시각화 편집, 저장/내보내기까지 한 흐름으로 진행할 수 있도록 만든 단일 HTML 기반 MVP입니다.

기존 시각화 도구들이 초보자에게 어렵게 느껴질 수 있다는 점에서 출발했습니다. 사용자가 업로드한 데이터 구조를 먼저 확인하고, 빈칸이나 숫자 형식처럼 차트 생성 전에 점검해야 할 부분을 살펴본 뒤, 적절한 차트를 선택하고 결과물을 편집할 수 있도록 설계했습니다.

Datawrapper의 데이터 확인 흐름과 Flourish의 차트 편집 경험을 참고했지만, 본 프로젝트는 캡스톤 수업 범위에서 구현한 독립적인 HTML 프로토타입입니다. 별도의 서버나 데이터베이스 없이 `mac_fin.html` 한 파일 안에서 주요 기능을 확인할 수 있도록 구성했습니다.

## 핵심 기능

### 1. CSV / Excel 업로드

- CSV, XLSX 파일 업로드
- 업로드된 데이터를 표 형태로 변환
- 한글 데이터 사용을 고려한 UI 구성

### 2. 데이터 확인 및 정리

- 업로드된 데이터 표 확인
- 숫자, 날짜, 문자, 빈칸 구분 표시
- 빈칸은 빨간 테두리로 강조
- 행 추가, 열 추가
- 행 삭제, 열 삭제
- 행과 열 바꾸기
- 변경 되돌리기
- 표 내부 가로/세로 스크롤

### 3. 차트 갤러리

- 막대 차트
- 레이싱 차트
- 그룹 막대
- 누적 막대
- 선 차트
- 영역 차트
- 도넛 차트
- 파이 차트
- 워드 클라우드
- 카드 지표
- 일부 차트는 준비 중 상태로 표시

### 4. Chart Studio 편집

- 차트 유형 변경
- 프로젝트 제목 입력
- 차트 제목 설정
- Top-N 표시 개수 조정
- 정렬 방식 변경
- 막대 간격, 라벨 영역, 투명도 조정
- 단색/팔레트 색상 모드
- 라벨, 범례, 축 관련 설정
- 원본 데이터 탭에서 X축/Y축 열 선택
- 선택한 열 하이라이트
- 실시간 미리보기

### 5. 저장 및 내보내기

- localStorage 기반 프로젝트 저장
- 내 프로젝트 페이지에서 저장된 프로젝트 카드 확인
- PNG 내보내기
- JPG 내보내기
- PDF 저장/인쇄
- SVG, PPTX, MP4, GIF는 향후 고도화 버전에서 제공 예정으로 표시

### 6. 튜토리얼 페이지

- 첫 사용자용 튜토리얼 페이지
- 파일 업로드부터 차트 편집까지 단계별 안내

## 사용 흐름

1. 새 시각화 만들기
2. CSV 또는 Excel 파일 업로드
3. 데이터 확인 및 정리
4. 차트 선택
5. Chart Studio에서 시각화 편집
6. 원본 데이터에서 X축/Y축 확인
7. 저장 또는 내보내기
8. 내 프로젝트에서 다시 열기

```mermaid
flowchart LR
    A[파일 업로드] --> B[데이터 확인 및 정리]
    B --> C[차트 선택]
    C --> D[Chart Studio 편집]
    D --> E[저장 및 내보내기]
    E --> F[내 프로젝트]
```

## 화면 구성

### 새 시각화 만들기

사용자는 파일 업로드 또는 직접 입력 흐름으로 새 시각화를 시작할 수 있습니다. 처음 사용하는 사용자를 위해 튜토리얼 페이지로 이동하는 안내도 제공합니다.

### 데이터 확인 및 정리

업로드된 표를 먼저 확인하는 단계입니다. 숫자, 날짜, 문자, 빈칸을 색상으로 구분해 초보자도 데이터가 어떻게 읽혔는지 빠르게 확인할 수 있습니다.

### 차트 갤러리

데이터 확인이 끝나면 원하는 차트 유형을 선택합니다. 현재 구현된 차트와 준비 중인 차트를 구분해 보여줍니다.

### Chart Studio

차트 제목, 색상, 정렬, Top-N, 라벨, 범례 등을 조정할 수 있는 편집 화면입니다. 원본 데이터 탭에서는 X축과 Y축으로 사용할 열을 선택하고, 선택한 열이 표에서 강조됩니다.

### 내 프로젝트

저장한 프로젝트를 카드 형태로 확인하고 다시 열 수 있습니다. 저장은 브라우저의 localStorage를 사용하므로 같은 브라우저 환경에서 프로토타입 동작을 확인할 수 있습니다.

## 현재 구현 범위

이 저장소의 구현은 캡스톤 수업용 HTML MVP 범위에 맞춰져 있습니다.

| 구분 | 현재 상태 |
| --- | --- |
| 파일 업로드 | CSV, XLSX 업로드 지원 |
| 데이터 확인 | 표 확인, 셀 타입 표시, 행/열 편집, 되돌리기 지원 |
| 차트 생성 | 주요 차트 유형 선택 및 Canvas 기반 렌더링 |
| Chart Studio | 제목, 색상, 정렬, Top-N, 라벨/범례 설정 지원 |
| 저장 | localStorage 기반 프로젝트 저장 |
| 내보내기 | PNG, JPG, PDF 저장/인쇄 지원 |
| 튜토리얼 | 첫 차트 만들기 안내 페이지 포함 |

## 향후 고도화 버전에서 제공 예정

현재 HTML 프로토타입에서 직접 구현하기 어려운 기능은 화면 안에서 준비 중으로 표시했습니다.

- SVG 내보내기
- PPTX 내보내기
- MP4 내보내기
- GIF 내보내기
- 계정 기반 저장 및 동기화
- 더 정교한 차트 렌더링과 애니메이션 내보내기
- 대용량 데이터 처리 최적화

## 실행 방법

별도의 설치 과정 없이 HTML 파일을 브라우저에서 열어 확인할 수 있습니다.

```text
mac_fin.html
```

저장 기능은 브라우저 localStorage를 사용합니다. 브라우저나 저장소 데이터를 초기화하면 저장된 프로젝트가 사라질 수 있습니다.

## 기술 구성

- HTML
- CSS / Tailwind CDN
- Vanilla JavaScript
- Canvas 기반 차트 렌더링
- localStorage 기반 프로젝트 저장
- SheetJS CDN을 활용한 Excel 파일 읽기

## 프로젝트 성격

MAC Capstone은 정식 서비스가 아니라 캡스톤 수업을 위한 단일 HTML 기반 MVP/프로토타입입니다. 핵심 목표는 사용자가 데이터를 올리고, 확인하고, 차트를 고르고, 편집하고, 저장/내보내기까지 이어지는 전체 사용자 경험을 한 파일 안에서 보여주는 것입니다.
