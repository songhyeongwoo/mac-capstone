데이터를 먼저 확인해요

차트를 만들기 전에 표가 올바르게 읽혔는지 확인하세요. 색으로 표시된 셀을 보면 숫자 날짜 문자 빈칸을 빠르게 점검할 수 있습니다.

1. 열 제목 확인

첫 줄이 열 이름이면 체크를 켜세요. 실제 데이터라면 체크를 끄면 됩니다.

2. 빈칸과 형식 확인

빨간 테두리는 비어 있는 값입니다. 파란 글자는 숫자, 초록 글자는 날짜로 인식됩니다.

3. 차트에 맞게 정리

검색, 정렬, 행·열 바꾸기, 행 추가, 열 추가로 표를 다듬은 뒤 차트를 선택하세요.

첫 번째 차트 만들기

파일을 올리고, 데이터를 확인하고, 차트를 고른 뒤 Chart Studio에서 다듬는 전체 과정을 따라 해보세요.

1

MAC 시작 화면에서 새 시각화 만들기

새 시각화 만들기 화면에서 직접 입력하거나 파일을 업로드할 수 있습니다. 처음 사용하는 경우 CSV 또는 Excel 파일을 올리는 방식이 가장 쉽습니다.

스크린샷 예정: 새 시각화 만들기 화면
2

CSV 또는 Excel 파일 업로드하기

파일 업로드 영역을 클릭하거나 파일을 드래그해 업로드합니다. 파일이 준비되면 ‘차트 자동 생성하기’를 눌러 다음 단계로 이동합니다.

스크린샷 예정: 파일 업로드 화면
3

데이터 확인 및 정리하기

업로드한 데이터가 표 형태로 표시됩니다. 열 제목, 빈칸, 숫자와 날짜 형식이 올바르게 인식되었는지 확인합니다.

파란 글자: 숫자 데이터
초록 글자: 날짜 데이터
검정 글자: 문자 데이터
빨간 테두리: 비어 있거나 확인이 필요한 값
스크린샷 예정: 데이터 확인 및 정리 화면
4

표를 차트에 맞게 다듬기

필요하면 행 추가, 열 추가, 행과 열 바꾸기, 정렬, 검색 기능을 사용해 차트에 맞게 표를 정리합니다. 차트가 이상하게 보일 때는 행과 열을 바꾸면 더 자연스럽게 보일 수 있습니다.

스크린샷 예정: 행과 열 바꾸기 기능
5

차트 갤러리에서 차트 선택하기

데이터 확인이 끝나면 차트 갤러리에서 원하는 차트 유형을 선택합니다. 막대 차트는 항목 비교에 좋고, 선 차트는 시간 흐름을 보여줄 때 좋습니다. 도넛/파이 차트는 전체 중 비중을 보여줄 때 적합합니다.

스크린샷 예정: 차트 갤러리 화면
6

Chart Studio에서 차트 편집하기

Chart Studio에서는 제목, 색상, 라벨, 정렬, 축 설정을 조정할 수 있습니다. 오른쪽 패널에서 차트 스타일을 바꾸면 중앙 미리보기 차트에 바로 반영됩니다.

스크린샷 예정: Chart Studio 편집 화면
7

원본 데이터에서 X축과 Y축 조정하기

원본 데이터 탭에서는 어떤 열을 X축과 Y축으로 사용할지 선택할 수 있습니다. X축은 기준이 되는 항목이고, Y축은 차트로 그릴 숫자 값입니다. 선택한 열은 표에서 색으로 강조되어 차트에 어떤 데이터가 들어가는지 쉽게 확인할 수 있습니다.

스크린샷 예정: 원본 데이터 X축/Y축 설정 화면
8

대시보드에 추가하고 저장하기

완성된 차트는 대시보드에 추가해 배치할 수 있습니다. 발표나 보고서에 활용할 수 있도록 저장하고 공유하는 흐름까지 이어집니다.

스크린샷 예정: 대시보드 추가 화면
